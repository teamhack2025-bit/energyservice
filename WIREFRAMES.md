# Wireframes & Layout Descriptions

## Design System Overview

### Color Palette
- **Primary Blue:** `#0066CC` - Main actions, links
- **Success Green:** `#00AA44` - Production, positive metrics
- **Warning Orange:** `#FF8800` - Warnings, alerts
- **Danger Red:** `#CC0000` - Consumption, errors
- **Neutral Gray:** `#6B7280` - Text, borders
- **Background:** `#F9FAFB` - Page background
- **Surface:** `#FFFFFF` - Cards, panels

### Typography Scale
- **H1:** 32px / 2rem - Page titles
- **H2:** 24px / 1.5rem - Section titles
- **H3:** 20px / 1.25rem - Subsection titles
- **Body:** 16px / 1rem - Default text
- **Small:** 14px / 0.875rem - Secondary text
- **Caption:** 12px / 0.75rem - Labels, metadata

### Spacing System
- Base unit: 4px
- Common: 8px, 16px, 24px, 32px, 48px, 64px

## Mobile Adaptations

### Sidebar â†’ Drawer
- Hamburger menu (â˜°) in header
- Slide-in drawer from left
- Overlay background
- Close on navigation or outside click

### Tables â†’ Cards
- Each row becomes a card
- Key data prominently displayed
- Actions as buttons within card
- Swipe actions (optional)

### Charts
- Maintain aspect ratio
- Touch-friendly tooltips
- Horizontal scroll if needed
- Full-screen view option

### Filters
- Collapse into "Filters" button
- Modal overlay on mobile
- Apply/Clear buttons
- Easy to dismiss

## Interaction Patterns

### Hover States (Desktop)
- Cards: Subtle shadow/elevation
- Buttons: Background color change
- Links: Underline or color change
- Charts: Tooltip appears

### Click States
- Buttons: Pressed/active state
- Cards: Navigate or expand
- Tabs: Active indicator
- Dropdowns: Open/close

### Loading States
- Skeleton loaders for content
- Spinner for buttons/actions
- Progress bars for long operations
- Optimistic updates where possible

### Error States
- Inline error messages
- Toast notifications
- Error boundaries for crashes
- Retry mechanisms

## Accessibility Considerations

### Focus Indicators
- Visible outline on focused elements
- Skip links for keyboard navigation
- Logical tab order

### Screen Reader Support
- ARIA labels on icons
- Descriptive alt text
- Live regions for dynamic content
- Semantic HTML structure

### Color Contrast
- Text meets WCAG AA (4.5:1)
- Interactive elements clearly visible
- Not relying solely on color for information

## Animation Guidelines

### Transitions
- Page transitions: 200ms ease
- Modal open/close: 300ms ease-out
- Hover effects: 150ms ease
- Loading: Smooth, non-jarring

### Micro-interactions
- Button press feedback
- Form validation feedback
- Success/error animations
- Progress indicators

## Implementation Notes

### Chart Libraries
- **Recommended:** Recharts or Chart.js
- Responsive by default
- Accessible (ARIA support)
- Customizable styling

### Date Pickers
- **Recommended:** react-datepicker or date-fns
- Mobile-friendly native pickers
- Clear date range selection
- Preset options

### Tables
- **Recommended:** TanStack Table (React Table)
- Virtualization for large datasets
- Sortable columns
- Responsive breakpoints

### Forms
- **Recommended:** React Hook Form
- Validation with Zod/Yup
- Accessible error messages
- Loading states
